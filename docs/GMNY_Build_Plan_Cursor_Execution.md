# GMNY Build Plan (Cursor + ChatGPT)

**Mission:** Build a secure USDC-powered cross-border payment platform enabling U.S. users to send money to Nigeria with direct Naira bank deposits.

**Objective:** Deliver an MVP demonstrating USD → USDC → Blockchain → Nigeria Liquidity Partner → NGN Bank Deposit.

**Technology:** Next.js, React, TypeScript, Tailwind CSS, NestJS, PostgreSQL, Prisma, Redis, BullMQ, Base, USDC, Circle APIs, Docker, GitHub, Railway, Vercel, Cloudflare.

**MVP Features:** Authentication, onboarding, KYC, bank linking, dashboard, transaction history, transfers, live FX, transfer tracking, admin portal, compliance dashboard, fraud monitoring, treasury analytics.

## Roadmap

- **Phase 1:** Repository, Cursor workspace, frontend/backend, database, authentication.
- **Phase 2:** Circle wallets, USDC transfers, blockchain engine.
- **Phase 3:** Nigerian payout integration, FX, settlement, notifications.
- **Phase 4:** KYC/AML, sanctions screening, audit logs, monitoring, production deployment.

**Compliance:** Design for KYC, AML, OFAC screening, transaction monitoring, audit trails, and licensed payment partners from day one.

**AI Workflow:** Use Cursor for development and ChatGPT as software architect, engineer, reviewer, QA, and technical writer. Each feature follows: Requirements → Design → Code → Review → Test → Deploy → Document.

**Vision:** Build the foundation for programmable cross-border financial infrastructure across Africa, expanding from remittances into B2B payments, payroll, treasury, and APIs.

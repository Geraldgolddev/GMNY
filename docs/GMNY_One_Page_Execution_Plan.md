# GMNY — One-Page Execution Plan

**Mission:** Build a USDC-powered U.S.-to-Nigeria payment platform that enables users to send USD and recipients to receive Naira quickly, securely, and at lower cost.

**MVP:** User signup, KYC, bank linking, USD→USDC conversion, blockchain transfer (Base/Polygon), NGN payout to Nigerian bank accounts, transaction tracking.

**Tech Stack:** Next.js, React, Tailwind, TypeScript, NestJS, PostgreSQL, Redis, Docker, Circle APIs, Base, Cloudflare.

**Revenue:** 0.5–1% FX spread, transfer fee ($2 or ~0.4%), business API subscriptions.

## Execution Roadmap

- **Week 1:** Authentication, database, wallets.
- **Week 2:** USDC deposits/transfers, transaction history.
- **Week 3:** Nigerian payout integration, FX quotes, dashboard.
- **Week 4:** KYC/AML, notifications, production deployment.

**Compliance:** Partner with licensed U.S. and Nigerian financial institutions for money movement, implement KYC, AML, sanctions screening, and transaction monitoring from day one.

**Long-Term Vision:** Expand from remittances into cross-border financial infrastructure supporting payroll, B2B payments, contractor payouts, treasury, and payment APIs across Africa.
